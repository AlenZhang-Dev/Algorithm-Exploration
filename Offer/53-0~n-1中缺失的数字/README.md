# 53. 0~n-1中缺失的数字

一个长度为n-1的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围0～n-1之内。在范围0～n-1内的n个数字中有且只有一个数字不在该数组中，请找出这个数字。

示例 1:

```
输入: [0,1,3]
输出: 2
```

示例 2:

```
输入: [0,1,2,3,4,5,6,7,9]
输出: 8
```


限制：

`1 <= 数组长度 <= 10000`

## 解析

排序数组搜索的问题，使用二分法解决。



## 题解

### Java

```java
class Solution {
    public int missingNumber(int[] nums) {
        int n = nums.length - 1, i = 0;
        while(i <= n){
            int m = (i + n) / 2;
            if(nums[m] == m)
                i = m + 1;
            else 
                n = m - 1;
        }
        return i;
    }
}
```
